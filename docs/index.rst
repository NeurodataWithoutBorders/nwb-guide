Welcome to the documentation for the NeurodataWithoutBorders (NWB) Graphical User Interface for Data Entry (GUIDE)!
=====================================================================================

..
  :scale: 100 %
  :align: right

The NWB GUIDE is an Electron-based application designed to navigate the complex process of converting data from common proprietary formats into the NWB
standard while fully complying with best practices expected of the Distributed Archives for Neurophysiology Data Integration (DANDI).
